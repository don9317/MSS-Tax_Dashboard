# MSS-Tax_Dashboard# MSS Tax Settings Dashboard

Simple dashboard prototype for configuring facility-level sales tax settings in My Sport Space (MSS).

## What This Does
- Allows facilities to:
  - Set sales tax rate
  - Select taxable categories
  - Control display settings
  - Acknowledge tax responsibility

## File Structure
- mss_tax_settings_dashboard.html → Main dashboard UI

## How to Use
1. Open the HTML file locally  
   OR  
2. Upload to GitHub and enable GitHub Pages

## Live Deployment (GitHub Pages)
1. Go to repo Settings  
2. Click "Pages"  
3. Select branch: main  
4. Save  

Your site will be live at:
https://yourusername.github.io/repository-name/
